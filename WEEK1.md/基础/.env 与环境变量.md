# .env 与环境变量

1. 编辑`.env`:

   ‍

   ```env
   APP_NAME=AI Chat Backend
   APP_VERSION=0.1.0
   TEST_API_KEY=hello_test_key
   ```
2. 添加代理地址：

   ```python
   PROXY_URL=http://127.0.0.1:7897 # 改成自己的
   ```
3. 添加大模型配置：

   ```python
   DEEPSEEK_API_KEY=你的真实API_KEY
   DEEPSEEK_BASE_URL=https://api.deepseek.com
   DEEPSEEK_MODEL=deepseek-chat
   ```
4. 创建配置文件：

   `app/core/config.py`写入：

   ```python
   import os

   from dotenv import load_dotenv


   load_dotenv()


   class Settings:
       app_name: str = os.getenv("APP_NAME", "AI Chat Backend")
       app_version: str = os.getenv("APP_VERSION", "0.1.0")

       test_api_key: str | None = os.getenv("TEST_API_KEY")
       proxy_url: str | None = os.getenv("PROXY_URL")

       deepseek_api_key: str | None = os.getenv("DEEPSEEK_API_KEY")
       deepseek_base_url: str = os.getenv(
           "DEEPSEEK_BASE_URL",
           "https://api.deepseek.com",
       )
       deepseek_model: str = os.getenv("DEEPSEEK_MODEL", "deepseek-v4-flash")


   settings = Settings()
   ```

   以后项目需要配置时统一调用：

   ```python
   settings.app_name
   settings.app_version
   settings.test_api_key
   ```
5. 在`main.py`中使用配置：

   修改：

   1. 开头添加：`from app.core.config import settings`
   2. 将`version`​修改为从环境变量中读取的：`version=settings.app_version,`
   3. 添加测试接口（不要直接返回 apikey，注意保护密匙安全）

      ```python
      @app.get("/config/test")
      def config_test():
          return {
              "app_name": settings.app_name,
              "app_version": settings.app_version,
              "has_test_api_key": settings.test_api_key is not None
          }
      ```
