# 安装依赖

1. Python 环境：

   ```bash
   python3 --version
   python3 -m pip --version
   which python3
   ```

2. 创建项目文件夹：

   ```bash
   mkdir ai-chat-backend #创建
   cd ai-chat-backend #跳转
   pwd #检查当前位置
   ```

3. 创建虚拟环境：

   ```bash
   python3 -m venv .venv #创建隐藏文件夹 .venv，就是项目自己的Python 环境
   ```

4. 激活虚拟环境：

   ```bash
   source .venv/bin/activate
   ```

5. 安装 FastAPI 相关依赖：

   ```bash
   python3 -m pip install fastapi uvicorn python-dotenv pydantic
   # fastapi：后端框架
   # uvicorn：启动 FastAPI 服务的工具
   # python-dotenv：读取 .env 环境变量
   # pydantic：做请求参数校验
   python3 -m pip freeze > requirements.txt # 将当前项目用到的依赖保存到 requirements.txt
   ```

6. 安装 httpx：

   ```bash
   python3 -m pip install httpx
   python3 -m pip freeze > requirements.txt

   # 检查是否安装成功（版本信息）
   python3 -m pip show httpx
   ```
