# 改进应用启动方式

> 为FastAPI应用设置启动任务，替代main.py中的init_db()，防止因依赖未加载完成就进行数据库初始化产生神秘错误
>
> **解决方法：使用lifespan**

1. `main.py`添加库：

   ```python
   from contextlib import asynccontextmanager

   from fastapi import FastAPI, HTTPException
   ```
2. 定义“程序的构造与析构函数”：

   ```python
   @asynccontextmanager # 将函数标记为上下文管理器
   async def lifespan(_: FastAPI):
       init_db()          #  这是“启动”时要执行的代码
       yield              # 这是“分隔线”，yield 上面是启动，下面是关闭
       # 以后可以在这里写关闭数据库连接的代码，比如 conn.close()
   ```
3. 在应用实例中添加这个函数：

   ```python
   app = FastAPI(
       title=settings.app_name,
       lifespan=lifespan,  # 把上面定义的开机流程，注册给 app
   )
   ```
