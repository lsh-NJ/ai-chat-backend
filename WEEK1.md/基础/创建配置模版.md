# 创建配置模版

> 真实环境配置文件`.env`无法上传仓库，因此需要一个模版文件来表示需要什么配置

1. 创建`.env.example`

   ```python
   touch .env.example
   ```
   写入：

   ```python
   APP_NAME=AI Chat Backend
   APP_VERSION=0.1.0

   DEEPSEEK_API_KEY=your_api_key_here
   DEEPSEEK_BASE_URL=https://api.deepseek.com
   DEEPSEEK_MODEL=deepseek-v4-flash

   DATABASE_PATH=data/app.db

   # Optional
   PROXY_URL=
   ```
