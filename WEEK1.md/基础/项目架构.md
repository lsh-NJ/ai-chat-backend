# 项目架构

1. 检查项目结构：

   `find app -maxdepth 3 -type f | sort`

   无视所有`__init__.py`后:

   ```python
   app/
   ├── main.py # 主程序
   ├── api/ 
   ├── core/
   │   └── config.py # 调取配置信息
   ├── db/
   │   └── database.py # 进行数据库操作
   ├── models/
   ├── schemas/
   │   ├── chat.py # 定义聊天结构体（类）
   │   └── conversation.py # 定义会话结构体（类）
   └── services/
       ├── external_api.py # 测试httpx获取接入外部api获取信息（可删除）
       └── llm_service.py # 大模型通信输出与流式输出
   ```
